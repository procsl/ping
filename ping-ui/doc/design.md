# 抽象组件设计

#### 组件设计
抽象组件使用`json schema`描述组件的功能骨架, 以实现布局与样式分离

1. 任意组件需要具有以下2个属性

    ```text
    String  id;     // 唯一Key,用来唯一标识一个组件类型, 在具体的实现中,可以用来指导具体的实现
    String  name;   // 名称, 用于展示或渲染
    ```

2. 组件具有事件属性,可以监听指定事件并注册
